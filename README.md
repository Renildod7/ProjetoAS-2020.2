<h1 align="center" >InforHub</h1>

#### O WIKInformation é uma rede social educacional, em que várias pessoas podem compartilhar informações sobre diversos temas. Além disso, um conjunto de usuários pode formar um grupo de pesquisa que pode trocar informações com outros grupos que se encaixam na mesma linha. Um usuário tem a possibilidade de contribuir para aumentar o grau de informação das publicações feitas por outros usuários. Cada publicação possui uma avaliação, a qual é dada pelos leitores.
