<h1 align="center" >WIKInformation</h1>

#### O WIKInformation é uma rede social educacional, em que várias pessoas podem compartilhar informações sobre diversos temas. Os usuários são divididos em três classes, administrador, pesquisador e usuário padrão. Sendo assim, algumas delas possuem restrições. Além disso, o usuário tem a possibilidade de contribuir para aumentar o grau de informação do artigo. Cada artigo possui uma avaliação, a qual é dada pelos leitores.
